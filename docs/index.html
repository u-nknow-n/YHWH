🚀 Início Rápido

1
Extraia o ZIP e execute o arquivo Vox Launcher.exe.
2
Configure o Game Directory (pasta do jogo) e o Cluster Directory (seu mundo).
3
Cole seu Server Token (veja instruções no app) e clique em LAUNCH!

💡 Dica: Confira o tutorial em vídeo do Jazzy Games no YouTube.

✨ Recursos Principais

Gestão de Mundo

Controles diretos para Salvar, Resetar, Rollback e Parar o servidor com um clique.

Console Integrado

Visualize logs em tempo real e execute comandos diretamente pelo launcher.

Monitoramento

Acompanhe contagem de jogadores, dia atual, estação e uso de memória RAM.

Suporte Avançado

Compatibilidade total com Mods e gerenciamento de múltiplos Shards (Fragmentos).

❓ Perguntas Frequentes

Por que o Windows Defender bloqueou o app?

Isso é um "falso positivo" comum em executáveis criados com Python. O Vox Launcher é seguro! Basta adicionar uma exceção no Windows Defender.

Como ativo os Mods no servidor?

Ative-os normalmente ao criar o mundo no jogo. O Launcher reconhecerá as configurações automaticamente.

Onde consigo o Cluster Token?

Gere um token na página oficial Klei Accounts.
